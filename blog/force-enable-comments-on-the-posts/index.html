Force enable comments on the posts

14 February 2020 | 1 min read

If you want to activate the comment feature on all posts then you can use this code.

You can use this code as a plugin or directly written in functions.php

function force_enable_comments() {
    global $wpdb;
    $wpdb->update( 
        $wpdb->posts, 
        array( 
            'comment_status' => 'open'
        ), 
        array(
            'post_type' => 'post'
        ), 
        array( 
            '%s'
        ), 
        array(
            '%s'
        ) 
    );
}
register_activation_hook( __FILE__, 'force_enable_comments' );

Download Plugin

Add to the plugin directory and activate it, the comment feature will be active in all posts
